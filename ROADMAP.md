# PolyStart - Beginner-Friendly Polymarket Copy Trading

**Problem:** 179+ Polymarket tools exist but ALL are for advanced traders. Beginners have no simple way to start.

**Solution:** A beginner-friendly companion that makes copy trading easy.

## Quick Start

### Prerequisites
- Python 3.11+
- Node.js 18+
- Telegram account

### Setup

```bash
# Backend
cd backend
pip install -r requirements.txt
python main.py

# Frontend  
cd frontend
npm install
npm run dev
```

## Features

- ✅ Curated top trader lists by category
- ✅ One-tap follow
- ✅ Simple Telegram alerts
- ✅ Beginner-friendly explanations
